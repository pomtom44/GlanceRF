Keyboard shortcuts

On the main dashboard you can press:

  • S – Open Setup (return here anytime)
  • L – Open Layout editor (add or rearrange cells)
  • M – Open Modules page
  • C – Open Config page

Shortcuts are ignored when the cursor is in a text field, so you can type normally in inputs.

To get back to Setup later, just press S from the main view.
